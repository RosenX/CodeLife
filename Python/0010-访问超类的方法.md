# 访问超类的方法

1. super是一个内置类（不是函数），用于访问超类的方法

2. 访问父类参数有几种写法：

```python
class Base:
    def say(self):
        print('I am base')

class Son(Base):
    def say(self):
        Base.say(self) # 旧式写法
        super(Son, self).say() # 新式写法。第二个如果省略，可以返回classmethod
        super().say() # super参数可省, python2中不可用
        print('I am son')

son = Son()
son.say()
```
